# 배송 상품과 물품 크기

## 배송 상품 종류

MOVB가 연동하는 배송 상품은 다음 네 가지입니다.

- QUICK: 일반 퀵 배송의 기본 선택입니다.
- QUICK_ECONOMY: 일반 퀵보다 저렴하고 여유 있게 보내는 선택입니다.
- QUICK_EXPRESS: 빠른 배송이 필요할 때 사용하는 급송 선택입니다.
- DOBO: 가까운 거리를 대상으로 하는 도보 배송 선택입니다.

실제 이용 가능 여부, 예상 시간과 가격은 출발지·도착지와 Sandbox 견적 응답에 따라
달라질 수 있습니다. 챗봇은 고정 가격을 추측하지 않고 견적 API 결과를 안내합니다.

## 물품 크기

물품 크기는 XS, S, M, L 네 단계로 입력합니다. MOVB 화면에서는 XS를 서류나 초소형
물품의 예시로 안내하지만, 실제 접수 전에는 사용 중인 API와 운영 기준에 맞는 크기를
선택해야 합니다.

## 차량 선택

퀵 주문에서는 차량을 자동 추천에 맡기거나 직접 지정할 수 있습니다. MOVB 챗봇과
접수 화면은 오토바이(MOTORCYCLE), 다마스(DAMAS), 라보(LABO), 1톤(TON)을
선택지로 제공합니다. 대형(L) 퀵은 차량 선택이 필수입니다. 도보 배송에는 차량을
지정하지 않습니다.

차량을 고른 뒤에도 실제 배차 가능 여부와 가격은 Kakao Mobility Sandbox의 견적
응답을 기준으로 안내합니다.

## 경유지 제약

현재 MOVB 모델에서는 경유지가 있는 주문을 QUICK과 카드 결제 조합으로 다룹니다.
DOBO와 QUICK_ECONOMY는 경유지를 포함할 수 없고, 경유지가 있는 주문은 카드 결제를
사용해야 합니다.
